## Space Travel 🌍
My first hands-on React project, built as a Frontend Mentor challenge while learning Tailwind CSS.

## 🧩 Overview
This project was developed as an introduction to React and Tailwind CSS as modern frontend tools. It focuses on building a foundation in React components and creating responsive layouts using Tailwind CSS.

## 🎯 Project Goals
The goal of this project was to take my first practical step into React development. I focused on understanding component structure, props flow, and basic state management, while also learning how to build modern and responsive layouts using Tailwind’s utility-first approach.

## 🛠 Tech Stack
- React  
- Tailwind 

## ✨ Main Features
- Reusable React components
- Responsive layouts built with Tailwind CSS
- Basic animation handling for smoother user interaction

## ⚠️ Challenges I Faced
- Learning the fundamentals of React, including component-based UI and props handling
- Limited knowledge of React Router, Context, and advanced concepts, making data flow management more difficult
- Learning Tailwind CSS and handling responsive design complexity
  
## 🤖 How I Used AI
- Used AI to better understand React basics such as components, props, and state, and to clarify how data flows between components
- Supported learning Tailwind CSS and building responsive layouts when documentation was not clear to my knowledge
- Assisted in fixing mistakes and understanding common beginner issues
